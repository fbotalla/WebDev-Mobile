CSS transforms and Transistions

We are now shiftin our focus to some interesting CSS features that the front-end development can do. I personally have never worked much with these features, therefore my learning process has been quite slow, but quite rewarding. Transforms and transitions allow us to have a more interactive GUI. We can place graphics anywhere we want and in whatever shape or form we want without effecting the other components. A good example of this would be the idea of having a pciture or object move throughout the screen This can be accomplished using CSS

Review the CSS code for more details on it. It's quite complex at first, but once thigs get going it becomes easier and eaiser to use. Here is a reference to the CSS.
